# typora-themeable-bear-config
Configuration for the Typora theme "themeable" to make it look like the Bear notes app.

1. Get Typoras "Themeable" Theme (here: https://theme.typora.io/theme/Typora-Themeable/ or here: https://github.com/jhildenbiddle/typora-themeable)
2. Open Typora settings
3. Go to "Appearance"
4. Click on "Open Theme Folder"
5. Put "themeable.user.css" there
